# LaysWebsite 🥔🔥

A cinematic, highly interactive landing page for Lay's Indian Magic Masala. Built with a modern React stack to deliver an explosive, sensory web experience.

## Tech Stack
- **React (Vite)**
- **Tailwind CSS v4**
- **GSAP (ScrollTrigger)** for high-performance canvas video scrubbing
- **Framer Motion** for buttery-smooth UI interactions

## Features
- 🎬 **Cinematic Hero**: Scroll-driven video sequence (240 frames) synced with kinetic typography.
- 🌶️ **Decoding the Flavor**: Interactive spices grid with dynamic Spice Meter.
- 📦 **Modern UI**: Glassmorphism cards, spring-animated accordions, and custom toast notifications.

## Run Locally
```bash
npm install
npm run dev
```
